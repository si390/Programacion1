Validando Formularios

Para validar formularios con JS necesitamos trabajar con eventos propios de formularios

Eventos

  • focus: se verifica al hacer foco sobre un campo del formulario.
  • blur: se verifica cuando el campo pierde foco, es decir cuando el cursor sale del campo en el que se encuentra
  • input: se verifica cuando cambia el valor de un elemento input, select o textarea
  • submit: se verifica en el momento del envío del formulario

¿Cómo sabemos si hay datos dentro de un campo input?

  • .value: averigua si están vacíos o no.

QueryStrings

Las peticiones por get permiten el uso de query strings

Definición: Es una cadena de texto que viaja en la url al momento de hacer una petición por get al servidor

Se escribe al final de la ruta comenzando con el signo "?" y está formada por pares "clave=valor" y deben estar separados por el caracter "&" si hay un par más de clave=valor

?search_query=digitalhouse

  • ? : inicio de la querystring
  • clave : query
  • = : operador de asignación
  • valor : digitalhouse

Enviando datos por querystrings

Hipervinculos

Con la etiqueta "a", tendremos que completar el href con los datos: ?id=35

Formulario

Con método GET

La clave será el atributo "name" de cada input

Obteniendo Objetos

OBJET location

Es un objeto literal con algunas propiedades que nos permiten obtener información de la url

location.search

Retorna la información en formato de cadena de texto, usando JavaScript podemos convertir la cadena de texto en un objeto literal con atributos y métodos para manipular los datos eficientemente

Objeto: URLSearchParams

Define métodos útiles para trabajar con los parámetros de una URL

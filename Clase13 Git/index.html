Trabajando con GIT

Vamos a tener GitHub como almacenamiento

  1. Crear repositorio remoto en GitHub
  2. Clonar repositorio remoto a repositorio local en mi PC
  3. Damos credenciales: Usuario y Email

Comando al trabajar en la terminal Git Bash

  1. Git add .
  2. Git commit "Digo que hice"
  3. Git push para subir los archivos

  • Git status: Chequeamos cómo se encuentra nuestro repositorio
  • Git clone: Clonar
